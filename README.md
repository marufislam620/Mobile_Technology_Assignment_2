#Mobile Technologies 11492 - Assignment 2

Student Name: Maruf Islam
Project Name: MT_assignment_2
Package Name: com.example.mt_assignment_2

This project was created in Android Studio using Java and XML.

Main features:
1. Barcode Reader using ML Kit Barcode Scanning
2. Content Reader using ML Kit Image Labeling
3. Text Reader using ML Kit Text Recognition
4. Firebase Realtime Database for saving text data
5. Internal storage for saving selected images
6. ListView showing saved analysed images
7. Detail screen with Edit, Delete, and Cancel

Tested on:
Pixel 9 Pro XL API 35 emulator

Firebase:
The app uses Firebase Realtime Database.
Database path: analysedImages
